# ESPECG

This project in intended to build a Holter Dynamic ECG Monitoring device with ESP32.

## Hardware

- ESP32
- SparkFun AD8232 ECG module
- ILI9341 240*320 TFT with XPT2046 Touch
- SD card

## Feature

- Real time ECG Data logging and display.

## Goals

- Real time ECG analysis and alarm.

- Data uploading.

- GUI.
