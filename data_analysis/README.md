# Data analysis for ESPECG

Since no easy to use open source ECG analysis program is ready for use. This project will try to create machine learning model and other utilities for real time ECG analysis.

The data analysis should be able to run on a server, or better, on a ESP32.

## Datasets

[PhysioBank](https://archive.physionet.org/physiobank/database/)

## Goals

- Heart Rate
- Heartbeat Classification
- Rhythm Classification
